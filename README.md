# constella

Autonomous AI agent payment system on the Stellar network via Soroban smart contracts.

See [PRD.md](./prd.md) for full product requirements.

## Structure

```
contracts/    — Soroban smart contracts (Rust)
frontend/     — Next.js dashboard (TypeScript + Tailwind)
relay/        — Event relay service (Cloudflare Workers)
scripts/      — Deployment and utility scripts
docs/         — Architecture and demo documentation
```
